## 練習問題

### Q1. `dw` / `dd` で削除する

`items = ["apple", "banana", "cherry", "REMOVE_ME"]` の行にカーソルを置いて：

1. `f"` → `;` `;` `;` で `"REMOVE_ME"` の `"` にジャンプしてください
2. `dw` で `"REMOVE_ME"` を削除してください（`, ` も残ることを確認）
3. `u` でアンドゥしてください
4. 今度は `di"` で `"..."` の中身だけを消してみてください
5. `u` でアンドゥしてください

> **確認**: `dw` は単語 + 後ろの空白を消します。`di"` は引用符の中身だけを消します。

---

### Q2. `D` と `dd` の違い

`unused_line = "この行は不要です"` の行にカーソルを置いて：

1. `dd` で行全体を削除してください
2. `u` でアンドゥしてください
3. 行の中ほどにカーソルを移動し（`w` で単語ジャンプ）、`D` で**カーソル以降**を削除してください
4. `u` でアンドゥしてください

---

### Q3. `cw` で単語を変更する

`def old_function(x: int) -> int:` の `old_function` にカーソルを置いて：

1. `cw` を押してください（`old_function` が消えて挿入モードになる）
2. `new_function` と入力してください
3. `Esc` でノーマルモードに戻ってください

> **ヒント**: `cw` = 削除 + 挿入モード移行。これを一番使います。

---

### Q4. `cc` で行全体を書き換える

`    return "placeholder"` の行にカーソルを置いて：

1. `cc` を押してください（行全体が消えて挿入モードになる）
2. `    return x * x` と入力してください
3. `Esc` で戻ってください

> **確認**: `cc` はインデントを保持したまま行全体を置き換えます。

---

### Q5. `ci"` と `ci(` でテキストオブジェクトを変更する（最頻出）

`greeting = "Hello, World!"` の行にカーソルを行内のどこかに置いて：

1. `ci"` を押してください（`"..."` の中身が消えて挿入モードになる）
2. `こんにちは、世界！` と入力してください
3. `Esc` で戻ってください

次に `result = calculate(100, 200)` の行で：

4. `ci(` を押してください（`(...)` の中身が消えて挿入モードになる）
5. `x, y` と入力してください
6. `Esc` で戻ってください

> **目標**: 設定ファイルや関数呼び出しの編集では `ci"` / `ci(` を反射的に使う。

---

### Q6. `yy` でコピーして `p` でペーストする

`def square(x: int) -> int:` の行にカーソルを置いて：

1. `yy` でその行をコピーしてください
2. `j` で1行下に移動してください
3. `p` でカーソルの**下**にペーストしてください
4. `P` でカーソルの**上**にペーストしてください（合計3行になる）
5. `2dd` で余分な2行をまとめて削除してください
6. `u` でアンドゥしてください

---

### Q7. ビジュアルモードで範囲を `y` / `d` する

`config = {` から始まる辞書ブロック（4行）にカーソルを置いて：

1. `V` で行ビジュアルモードに入ってください
2. `3j` で4行選択してください
3. `y` でヤンクしてください（ビジュアルが解除される）
4. ファイル末尾に移動（`G`）して `p` でペーストしてください
5. `u` でアンドゥしてください

---

### Q8. 総合チャレンジ

以下の変更を**最小のキー操作**で行ってください：

1. `process_data(input_data, verbose=True)` の呼び出しを `process_data(result, verbose=False)` に変更する
   - ヒント: `ci(` で括弧の中を一気に変える
2. `old_name = "legacy"` を `current_name = "latest"` に変更する
   - ヒント: `ciw` を2回
3. コメント行 `# TODO: remove this later` を `dd` で削除する

## 練習用ファイル

`practice/03_編集の基本パターン_practice.py` を作成してください：

```python
# 練習用ファイル: 03_編集の基本パターン_practice.py


def old_function(x: int) -> int:
    """2乗を返す（関数名が古い）。"""
    return "placeholder"


def calculate(a: int, b: int) -> int:
    """合計を計算する。"""
    result = calculate(100, 200)
    return a + b


greeting = "Hello, World!"
old_name = "legacy"
unused_line = "この行は不要です"

items = ["apple", "banana", "cherry", "REMOVE_ME"]

config = {
    "debug": True,
    "timeout": 30,
    "retries": 3,
}


def process_data(input_data, verbose=True):
    """データを処理する。"""
    if verbose:
        print(f"処理中: {input_data}")
    return input_data


# TODO: remove this later
if __name__ == "__main__":
    process_data(input_data, verbose=True)
```
