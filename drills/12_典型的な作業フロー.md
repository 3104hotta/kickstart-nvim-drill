## 練習問題

### フロー1. 関数を修正する

実際のコード修正を想定した一連の操作をキーボードのみで行ってください。

**シナリオ**: `calculate_tax` 関数の引数に `region: str` を追加し、内部ロジックを変更する。

1. `<leader>sf` で `practice/12_作業フロー_practice.py` を開いてください
2. `/calculate_tax` で関数を検索してください
3. `gd` で定義行に移動してください（またはそのまま `n` で）
4. `K` でシグネチャを確認してください
5. `ci(` で引数リスト全体を変更し、`amount: float, region: str = "JP"` に書き換えてください
6. 関数本体の `return amount * 0.1` を `cc` で `return amount * (0.1 if region == "JP" else 0.08)` に書き換えてください
7. `:w` で保存してください

---

### フロー2. プロジェクト内の文字列を一括変更する

**シナリオ**: `legacy_api` という名前を `modern_api` に変更する。

1. `<leader>sw` で（または `*` で）`legacy_api` を検索してください
2. `cgn` で最初の一致を `modern_api` に変更してください
3. `Esc` → `.` で次の一致を変更してください
4. 変更したくない箇所は `n` でスキップしてください
5. 全体置換なら `:%s/legacy_api/modern_api/gc` でも OK です

### フロー3. エラーを修正する

**シナリオ**: 型エラーと未使用変数の警告を修正する。

1. `]e` でエラー箇所へジャンプしてください
2. `<leader>e` でエラーメッセージを確認してください
3. `<leader>ca` でコードアクション（自動修正）が使えるか試してください
4. 手動で修正が必要な場合は `ci(` / `cw` / `cc` を使って修正してください
5. `[e` / `]e` で全エラーを巡回して修正してください
6. `<leader>q` で quickfix リストを開き、残りのエラーを確認してください

### フロー4. ファイルを並べて比較・編集する

**シナリオ**: 古い実装と新しい実装を見比べながら移行する。

1. `:vsp` でウィンドウを垂直分割してください
2. 右ウィンドウで `<leader>sf` を使って `practice/03_編集の基本パターン_practice.py` を開いてください
3. `Ctrl-h` / `Ctrl-l` で2つのウィンドウを行き来してください
4. `Ctrl-w =` でサイズを均等にしてください
5. 左のファイルの内容を参考にしながら、右のファイルに変更を加えてください
6. `:wa` で両方のファイルを保存してください

### フロー5. 新機能を追加する（総合）

`practice/12_作業フロー_practice.py` に `calculate_discount` 関数を追加してください。

**要件**:
- 引数: `price: float, discount_rate: float = 0.0`
- 戻り値: `float`（割引後の価格）
- `discount_rate` は 0〜1 の範囲でなければならない（範囲外は `ValueError`）

**使うべき操作**:
- `G` でファイル末尾へ移動
- `o` で新しい行を追加して関数を記述
- `ci(` で引数を書き換える練習
- `:w` で保存

## 練習用ファイル

`practice/12_作業フロー_practice.py` を作成してください：

```python
# 練習用ファイル: 12_作業フロー_practice.py


def calculate_tax(amount: float) -> float:
    """税額を計算する。"""
    return amount * 0.1


def calculate_total(amount: float) -> float:
    """税込み合計を計算する。"""
    tax = calculate_tax(amount)
    return amount + tax


def legacy_api(endpoint: str, data: dict) -> dict:
    """旧APIを呼び出す。"""
    print(f"[legacy_api] {endpoint}: {data}")
    return {"status": "ok"}


def fetch_user(user_id: int) -> dict:
    """ユーザー情報を取得する。"""
    result = legacy_api(f"/users/{user_id}", {})
    return result


def update_user(user_id: int, data: dict) -> dict:
    """ユーザー情報を更新する。"""
    result = legacy_api(f"/users/{user_id}", data)
    return result


def main():
    total = calculate_total(1000)
    print(f"合計: {total}")

    user = fetch_user(1)
    update_user(1, {"name": "新しい名前"})
    print(user)


if __name__ == "__main__":
    main()
```
