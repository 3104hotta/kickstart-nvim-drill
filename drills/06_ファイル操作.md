## 準備

この章は実際に Neovim を操作しながら確認します。まず練習用ファイルを開いてください：

```
nvim practice/06_ファイル操作_practice.py
```

---

## 練習問題

### Q1. 保存する

`practice/06_ファイル操作_practice.py` を開いた状態で：

1. ファイルの末尾に `# 変更を加えました` と追記してください（`G` → `o` → 入力 → `Esc`）
2. `:w` で保存してください
3. `:w practice/06_copy.py` で別名で保存してください

> **確認**: `:w` はカレントファイル、`:w {name}` は別名保存です。

---

### Q2. ファイルを再読み込みする

1. ターミナルの別タブで `echo "# 外部追記" >> practice/06_ファイル操作_practice.py` を実行してください（または後で試してください）
2. Neovim 内で `:e!` を実行してください
3. ファイルの内容が更新されていることを確認してください

> **ヒント**: CI/CD で生成されたファイルや git pull 後のファイルを再読み込みするときに使います。

---

### Q3. 別のファイルを開く

現在のバッファで：

1. `:e practice/03_編集の基本パターン_practice.py` を実行して別ファイルを開いてください
2. `Ctrl-^` を押してください（直前のバッファ、つまり `06_ファイル操作_practice.py` に戻る）
3. もう一度 `Ctrl-^` でまた切り替えてください

> **目標**: ファイル間の行き来に `:e` と `Ctrl-^` を使う。

---

### Q4. 保存して閉じる / 閉じる

1. 現在のファイルを `:wq` で保存して閉じてください
2. 再び `nvim practice/06_ファイル操作_practice.py` で開いてください
3. 今度は変更せずに `ZZ` で閉じてください（`:wq` と同じ）
4. 再び開いて、何か変更してから `:q!` で**保存せずに**閉じてください
5. ファイルが変更前のままであることを確認してください

---

### Q5. 複数ファイルを同時に開いて全バッファを保存する

1. `nvim practice/06_ファイル操作_practice.py practice/03_編集の基本パターン_practice.py` で2ファイルを同時に開いてください
2. `:ls` で開いているバッファ一覧を確認してください
3. `:b2` でバッファ2に切り替えてください
4. それぞれのファイルに軽微な変更を加えてください
5. `:wa` で全バッファを一括保存してください

---

### Q6. `ZQ` で強制終了する

1. ファイルに変更を加えてください
2. `:q` を試してみてください（変更があると怒られる）
3. `ZQ` で変更を破棄して強制終了してください

> **確認**: `ZQ` は `:q!` と同じです。間違えて変更した内容を捨てるときに使います。

---

### Q7. 全ウィンドウを閉じる

1. `:vsp` でウィンドウを分割してください
2. `:qa` を実行してください（全ウィンドウが閉じられる）
3. 変更がある場合は `:qa!` で強制終了できます

---

## 練習用ファイル

`practice/06_ファイル操作_practice.py` を作成してください：

```python
# 練習用ファイル: 06_ファイル操作_practice.py
# この章ではファイルの保存・開閉・切り替えを練習します


def main():
    print("ファイル操作の練習")
    print("保存: :w")
    print("保存して閉じる: :wq または ZZ")
    print("保存せずに閉じる: :q! または ZQ")
    print("全バッファを保存: :wa")
    print("再読み込み: :e!")


if __name__ == "__main__":
    main()
```
