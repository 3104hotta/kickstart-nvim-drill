## 練習問題

### Q1. `gcc` でコメントをトグルする

1. `result = process(data)` の行にカーソルを置いてください
2. `gcc` を押してください（コメントになる）
3. もう一度 `gcc` を押してください（コメントが外れる）
4. 別の行でも試してください

---

### Q2. `gc` + モーションでまとめてコメントする

`COMMENT_BLOCK` セクションの関数 `slow_algorithm` にカーソルを置いて：

1. `gcip` を押してください（段落全体をコメントトグル）
2. `u` でアンドゥしてください
3. `V` で数行選択してから `gc` を押してください（選択範囲をコメントトグル）
4. `u` でアンドゥしてください

---

### Q3. `>>` / `<<` でインデントを調整する

`INDENT_SECTION` のブロックで：

1. `misaligned_code_a` の行にカーソルを置いてください
2. `>>` を押してください（右にインデント）
3. `<<` を押してください（左にインデント）
4. `V` + `2j` で3行選択し、`>` でインデント、`<` で戻してください

---

### Q4. `=G` で自動インデントを適用する

1. `BROKEN_INDENT` セクションのインデントが壊れた行に移動してください
2. `=G` を押してください（カーソルからファイル末尾まで自動インデントが適用される）
3. `u` でアンドゥしてください

> **注意**: `=G` は LSP のフォーマッタではなく Vim 組み込みのインデントを使います。Python では予期しない動作をすることがあります。

---

### Q5. `Ctrl-a` / `Ctrl-x` で数値を増減する

1. `MAX_RETRIES = 3` の `3` にカーソルを置いてください
2. `Ctrl-a` を押してください（`4` になる）
3. `Ctrl-a` をもう一度押してください（`5` になる）
4. `Ctrl-x` を押してください（`4` に戻る）
5. `10Ctrl-a` を押してください（`14` になる）

---

### Q6. `~` で大文字・小文字を切り替える

1. `status = "active"` の `active` の先頭にカーソルを置いてください
2. `~` を数回押して `ACTIVE` に変換してください
3. `u` でアンドゥしてください
4. `gUiw` でカーソル下の単語全体を大文字にしてみてください
5. `guiw` で小文字に戻してください

---

### Q7. `J` で行を結合する

1. `first_part =` の行にカーソルを置いてください
2. `J` を押してください（次の行が現在行に結合される）
3. `u` でアンドゥしてください
4. `3J` を押してください（次の2行を結合して1行にする）
5. `u` でアンドゥしてください

---

### Q8. quickfix リストを操作する

1. `<leader>q` を押してください（全診断を quickfix に）
2. `:copen` で quickfix リストを開いてください（既に開いていれば不要）
3. `:cn` で次の項目へ移動してください
4. `:cp` で前の項目へ戻ってください
5. `:cclose` で quickfix を閉じてください

---

### Q9. 挿入モード中に `Ctrl-o` で一時的にコマンドを実行する

1. 任意の行で `i` で挿入モードに入ってください
2. `Ctrl-o` を押してください（一時的にノーマルモードコマンドを1つ実行できる）
3. `$` を押してください（行末に移動し、自動的に挿入モードに戻る）
4. これを使うと挿入中に `Ctrl-o w` で単語単位移動、`Ctrl-o dd` で行削除などができます

---

### Q10. 総合チャレンジ

以下を最小キー操作で行ってください：

1. `TIMEOUT = 30` の `30` を `Ctrl-a` で `60` に増やしてください（`30Ctrl-a`）
2. `flag = false` の `false` を `gUiw` で `FALSE` にしてください
3. 3行あるリスト内容をそれぞれ `gcc` でコメントアウトしてください（`.` を活用）
4. コメントアウトした行を `V` で選択して `gc` でまとめて元に戻してください

---

## 練習用ファイル

`practice/11_その他のコマンド_practice.py` を作成してください：

```python
# 練習用ファイル: 11_その他のコマンド_practice.py


MAX_RETRIES = 3
TIMEOUT = 30
flag = false


# --- Q2 用: COMMENT_BLOCK ---
def slow_algorithm(data: list) -> list:
    result = []
    for item in data:
        result.append(item * 2)
    return result


# --- Q3 用: INDENT_SECTION ---
misaligned_code_a = 1
misaligned_code_b = 2
misaligned_code_c = 3


result = process(data)
status = "active"


# --- Q7 用: 行結合 ---
first_part = (
    "これは長い文字列の"
    "前半部分です"
)


# --- Q10 用: コメントトグル ---
todo_list = [
    "タスク1: ドキュメントを書く",
    "タスク2: テストを追加する",
    "タスク3: レビューを依頼する",
]


# --- Q4 用: BROKEN_INDENT ---
def broken_function():
    x = 1
    y = 2
    if x:
        z = x + y
    return z
```
