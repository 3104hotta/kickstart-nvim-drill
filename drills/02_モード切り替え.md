## 練習問題

### Q1. `i` と `a` の使い分け

`name = "Alice"` の行にカーソルを置いて：

1. `f"` で最初の `"` にジャンプし、`a` で `"` の**後ろ**から挿入モードに入り `Hello, ` と入力してください
2. `Esc` でノーマルモードに戻ってください
3. `f"` で2番目の `"` にジャンプし、`i` で `"` の**前**から挿入モードに入り ` san` と入力してください
4. `Esc` で戻ってください

> **確認**: `i` はカーソルの左、`a` はカーソルの右から入力が始まります。

---

### Q2. `I` と `A` で行頭・行末に追記

`score = 100` の行にカーソルを置いて：

1. `A` を押して行末から挿入モードに入り、`  # 満点` と追加してください
2. `Esc` で戻ってください
3. `I` を押して行頭から挿入モードに入り、`# ` と追加してください
4. `Esc` で戻り、`:u` でアンドゥして元に戻してください

> **目標**: 行頭・行末への追記に `0i` / `$a` は使わず `I` / `A` を使う。

---

### Q3. `o` と `O` で新しい行を追加

`def greet(name: str) -> str:` の行にカーソルを置いて：

1. `o` を押して**下**に新しい行を追加し、`    print("呼ばれました")` と入力してください
2. `Esc` で戻ってください
3. `:u` でアンドゥしてください
4. `O` を押して**上**に新しい行を追加し、`# 挨拶関数` と入力してください
5. `Esc` で戻ってください

> **確認**: `o` / `O` は新行追加 + 挿入モード移行を1キーで行います。

---

### Q4. ビジュアルモードで範囲選択

`items = ["apple", "banana", "cherry"]` の行にカーソルを置いて：

1. `v` でビジュアルモードに入り、`w` を数回押して `"apple"` の部分を選択してください
2. `Esc` でキャンセルしてください
3. `V` で行ビジュアルモードに入り、`j` を2回押して3行選択してください
4. `Esc` でキャンセルしてください

---

### Q5. 矩形ビジュアル（`Ctrl-v`）で複数行を同時編集

以下の3行（`practice` ファイル内の `INDENT_BLOCK` セクション）にカーソルを置いて：

```
apple = 100
banana = 200
cherry = 300
```

1. `apple` の `a` にカーソルを合わせてください
2. `Ctrl-v` で矩形ビジュアルに入り、`2j` で3行分選択してください
3. `I` を押して `# ` と入力し、`Esc` を押してください
4. 3行すべての行頭に `# ` が追加されたことを確認してください
5. `u` でアンドゥして元に戻してください

> **ヒント**: 矩形ビジュアル中の `I` は選択したすべての行に同じ文字を挿入します。

---

### Q6. 総合チャレンジ

`practice/02_モード切り替え_practice.py` を開き、以下の変更を**最小のキー操作**で行ってください：

1. `greet` 関数の `return` 行を `return f"こんにちは、{name}さん！"` に書き換える（`cc` は第3章なので、ここでは `I` や `A` を駆使してください）
2. `score = 100` の行の下に `level = "S"` という新しい行を `o` で追加する
3. `items` リストの3行上に `# フルーツリスト` というコメントを `O` で追加する

## 練習用ファイル

`practice/02_モード切り替え_practice.py` を作成してください：

```python
# 練習用ファイル: 02_モード切り替え_practice.py
# このファイルを nvim で開いてモード切り替えを練習してください


name = "Alice"
score = 100


def greet(name: str) -> str:
    """挨拶文を返す。"""
    return f"Hello, {name}"


items = ["apple", "banana", "cherry"]


# --- INDENT_BLOCK（Q5用） ---
apple = 100
banana = 200
cherry = 300
# --- END ---
```
