## 練習問題

### Q1. 単語単位の移動

ファイルを開いたら、カーソルは1行目の先頭にあります。

1. `w` を押して単語ごとに右へ進み、`calculate_total` の `total` の先頭に移動してください
2. `b` を押して1単語分戻ってください
3. `e` を押して、現在の単語の末尾に移動してください

> **確認**: `e` を押したとき、カーソルは単語の最後の文字の上にありますか？

---

### Q2. 行頭・行末への移動

`calculate_total` 関数の定義行（`def calculate_total(prices: list[float]) -> float:`）にカーソルを置いた状態で：

1. `$` を押して行末に移動してください
2. `0` を押して行頭（列0）に戻ってください
3. `^` を押してください。`0` との違いを確認してください

> **確認**: `0` は列0、`^` は最初の非空白文字。インデントされた行では挙動が異なります。

---

### Q3. ファイル先頭・末尾への移動

1. `G` を押してファイルの末尾行に移動してください
2. `gg` を押してファイルの先頭に戻ってください
3. `50G`（数字 + G）を押してみてください。何が起きましたか？

> **ヒント**: `{行番号}G` で指定行へ直接ジャンプできます。

---

### Q4. 半画面スクロール（最重要）

1. `Ctrl-d` を数回押してファイルを下にスクロールしてください
2. `Ctrl-u` を数回押して上に戻ってください
3. `h` / `j` / `k` / `l` は今後なるべく使わず、`Ctrl-d` / `Ctrl-u` + `w` / `b` での移動を習慣にしてください

> **目標**: 1画面以上の移動に矢印キーや `jjjjj...` を使わない。

---

### Q5. 段落ジャンプ

ファイルには空行で区切られた複数のブロックがあります。

1. `}` を押して次の空行（段落の終わり）へジャンプしてください
2. `{` を押して前の段落に戻ってください
3. `}` を連打してファイル末尾まで移動してください

> **確認**: 関数と関数の間の空行が「段落の区切り」になっていますか？

---

### Q6. 対応する括弧へのジャンプ

`calculate_total(prices: list[float]) -> float:` の `(` にカーソルを合わせて：

1. `%` を押して対応する `)` に移動してください
2. もう一度 `%` を押して元の `(` に戻ってください
3. `[` や `{` でも同様に試してください

---

### Q7. 行内文字検索

`result = sum(price * (1 - discount) for price in prices)` の行にカーソルを置いて：

1. `f(` を押して最初の `(` にジャンプしてください
2. `;` を押してください（同じ方向の次の一致へ）
3. `F)` を押して行内で手前の `)` に戻ってください

> **ヒント**: `f` のあと `;` で次、`,` で前の一致へ移動できます。

---

### Q8. 総合チャレンジ

**タイム計測推奨。キー数を最小にして移動してみましょう。**

ファイル先頭から出発して、以下をすべてキーボードのみで行ってください：

1. `apply_discount` 関数の定義行に移動する（`/` 検索は使わず、`G` / `gg` / `{` / `}` / `w` / `b` のみ）
2. その行の `discount` という単語の先頭にカーソルを置く
3. `%` で対応する括弧を確認する
4. `Ctrl-d` でファイル末尾付近まで移動する
5. `gg` で先頭に戻る

## 練習用ファイル

`practice/01_移動_practice.py` を作成してから始めてください。

```python
# 練習用ファイル: 01_移動_practice.py
# このファイルを nvim で開いて移動コマンドを練習してください


def calculate_total(prices: list[float]) -> float:
    """商品の合計金額を計算する。"""
    result = sum(price for price in prices)
    return result


def apply_discount(price: float, discount: float) -> float:
    """割引率を適用した価格を返す。"""
    if discount < 0 or discount > 1:
        raise ValueError("割引率は0以上1以下である必要があります")
    result = sum(price * (1 - discount) for price in [price])
    return result


def format_price(amount: float, currency: str = "JPY") -> str:
    """金額を通貨記号付きの文字列にフォーマットする。"""
    if currency == "JPY":
        return f"¥{amount:,.0f}"
    elif currency == "USD":
        return f"${amount:,.2f}"
    else:
        raise ValueError(f"未対応の通貨: {currency}")


def generate_receipt(items: list[dict], discount: float = 0.0) -> str:
    """領収書の文字列を生成する。"""
    lines = []
    for item in items:
        name = item["name"]
        price = item["price"]
        discounted = apply_discount(price, discount)
        lines.append(f"  {name}: {format_price(discounted)}")
    total = calculate_total([apply_discount(i["price"], discount) for i in items])
    lines.append(f"合計: {format_price(total)}")
    return "\n".join(lines)


# エントリーポイント
if __name__ == "__main__":
    items = [
        {"name": "りんご", "price": 150.0},
        {"name": "バナナ", "price": 80.0},
        {"name": "みかん", "price": 120.0},
    ]
    receipt = generate_receipt(items, discount=0.1)
    print(receipt)
```
